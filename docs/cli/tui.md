# `shittimchest tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
shittimchest tui
shittimchest tui --url ws://127.0.0.1:18789 --token <token>
shittimchest tui --session main --deliver
```
