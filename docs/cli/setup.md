# `shittimchest setup`

Initialize `~/.shittimchest/shittimchest.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
shittimchest setup
shittimchest setup --workspace ~/.shittimchest/workspace
```

To run the wizard via setup:

```bash
shittimchest setup --wizard
```
